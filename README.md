SystemDiag
----------
SystemDiag es una herramienta de diagnóstico de sistemas diseñada para proporcionar una visión completa y detallada sobre el rendimiento y estado de tu equipo. Desarrollado por David Álvarez de Greenlabs, este script permite a los usuarios analizar componentes clave como CPU, RAM, disco duro, GPU, y mucho más.

🚀 Características
Monitoreo de hardware: Evalúa en tiempo real el estado de la memoria RAM y el CPU.
Salud del disco duro: Información detallada sobre la salud y temperatura de los discos duros.
Alertas automáticas: Recibe notificaciones cuando se detectan condiciones críticas, como altas temperaturas o uso excesivo de recursos.
Compatibilidad: Funciona en sistemas basados en Unix (Linux, macOS).
Informes detallados: Genera diagnósticos completos para una mejor comprensión del estado del sistema.
📦 Requisitos
Python 3.x
Librerías estándar de Python (no requiere dependencias externas).
⚙️ Instalación
Clona el repositorio:
git clone https://github.com/lemxmbx/Systemdiag.git
Navega al directorio del proyecto:
cd Systemdiag
Ejecuta el script:
python3 systemdiag.py
🛠️ Uso desde USB
SystemDiag está diseñado para ser portable. Puedes ejecutarlo directamente desde un USB, sin necesidad de instalar nada en el equipo donde lo estés utilizando.
